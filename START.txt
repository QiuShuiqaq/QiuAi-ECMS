QiuAi-ECMS 启动说明

一、项目位置
项目根目录：
`D:\Workspace\QiuAi-ECMS`

二、首次启动
1. 打开 PowerShell
2. 进入项目目录：
`cd D:\Workspace\QiuAi-ECMS`
3. 如未安装依赖，执行：
`npm install`
4. 启动项目：
`npm run dev`

三、日常启动
以后每次启动只需要执行：
`cd D:\Workspace\QiuAi-ECMS`
`npm run dev`

四、启动后会发生什么
1. Vite 前端开发服务会启动，默认端口为 `5173`
2. Electron 桌面窗口会自动打开
3. 关闭当前命令行窗口，会同时停止开发服务

五、如果 Electron 窗口没有自动弹出
先确认 `http://127.0.0.1:5173` 能访问，然后在项目根目录执行：
`cmd /c "set ELECTRON_RUN_AS_NODE=& set NODE_ENV=development& set VITE_DEV_SERVER_URL=http://127.0.0.1:5173& .\node_modules\electron\dist\electron.exe ."`

六、常用命令
启动开发环境：
`npm run dev`

仅构建前端：
`npm run build:renderer`

运行测试：
`npm test`

检查代码：
`npm run lint`

七、注意事项
1. 当前正式项目是 `QiuAi-ECMS`，不要再改 `QiuAi`
2. 用户本地数据目录默认在：
`C:\Users\admin\AppData\Roaming\QiuAi-ECMS`
3. 选品页面有本地缓存和安全延迟，不是每次切换都会立刻请求源站
4. 当前仓库里有不少未提交改动，继续开发前先看 `git status`
